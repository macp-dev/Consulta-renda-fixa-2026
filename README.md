# PROJETO: consulta-renda-fixa-2026

**Ferramenta de análise e simulação para investidores em fase de usufruto — com foco em segurança, previsibilidade de fluxo de caixa e sustentabilidade do patrimônio.**


## 🎯 Objetivo

Este projeto foi desenvolvido para auxiliar investidores que já concluíram a fase de acumulação de patrimônio e agora buscam:

- **Garantir uma aposentadoria sustentável**;
- **Realizar saques mensais sem comprometer o crescimento real do patrimônio**;
- **Aplicar com segurança**, priorizando ativos de renda fixa adequados ao perfil conservador e à necessidade de previsibilidade.


## 🧠 O que o projeto faz

A partir de um conjunto de fontes organizadas no **NotebookLM**, o sistema permite:

- Simular diferentes estratégias de saque mensal;
- Avaliar o impacto da inflação e da tributação;
- Comparar cenários com diferentes alocações em renda fixa (pós-fixados, prefixados, atrelados à inflação);
- Monitorar indicadores de longevidade do patrimônio.
